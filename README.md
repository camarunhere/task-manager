# Task Manager CLI Application

## Description

This is a simple command-line interface (CLI) application to manage tasks. The application allows users to add, view, delete, and mark tasks as complete. Tasks are saved to and loaded from a `tasks.json` file.

## How to Run

1. Ensure you have Python installed on your system.
2. Clone or download this repository.
3. Navigate to the `task_manager` directory.
4. Run the application using the command:
   ```bash
   python task_manager.py
